# GitHub Actions CI Demo

A minimal test project with a CI pipeline using GitHub Actions.

## Getting Started

```bash
git clone https://github/kiefertaylorland/github-actions-ci-for-tests
cd github-actions-ci-for-tests
npm install
npm test
```
